# Shortify AI 🚀

Shortify AI is a modern AI-powered URL Shortener SaaS built using the MERN Stack.  
It allows users to create smart short URLs, generate AI-powered shortnames, manage links, and interact with an integrated AI assistant.

# 💡 Why I Built This

While working on my laptop, I often had dozens of important tabs open for coding, productivity, learning, and development tools.

Closing them felt inconvenient because reopening all those resources later consumed time and broke workflow continuity.

So I built **Shortify AI** — a smart AI-powered URL management platform where I can:
- store useful links
- organize resources
- generate memorable shortnames
- quickly access everything from one dashboard

This project started as a personal productivity solution and evolved into a full SaaS-style platform.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/shortify-ai.git
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
npm install
node server.js
```
